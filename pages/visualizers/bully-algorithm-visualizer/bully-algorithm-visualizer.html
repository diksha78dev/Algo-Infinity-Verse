👑 How the Bully Algorithm Works

📡

Detecting Failure

Nodes periodically ping the coordinator. When a ping times out, the node concludes the coordinator is down and starts an election — no central authority tells anyone this happened.

📢

The Election Message

The node that noticed the failure sends ELECTION to every node with a HIGHER id. If any of them respond, they take over the election and the original node steps back — "bullied" out by a bigger node, hence the name.

🏆

Declaring Victory

If a node sends ELECTION to all higher IDs and gets no response, it means it's the highest surviving node. It declares itself coordinator and broadcasts COORDINATOR to everyone.

🔀

Concurrent Elections Still Converge

Multiple nodes can notice the failure and start elections at the same time. Because every election independently discovers and defers to the highest alive ID, they all converge to the exact same winner regardless of timing.
