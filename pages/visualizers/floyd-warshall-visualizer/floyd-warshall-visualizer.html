How Floyd-Warshall Works

1

Dynamic Programming

Floyd-Warshall builds the shortest paths incrementally. In phase k, it checks if the path going through vertex k is shorter than the currently known shortest path.

DP O(V³) Time
2

Distance Matrix

It maintains a 2D matrix where dist[i][j] stores the shortest distance from node i to j. Initially, it holds direct edge weights.

Adjacency Matrix O(V²) Space
3

Relaxation Rule

For every pair (i, j), the algorithm tests: dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]). If going through k is cheaper, it updates.

Relaxation Transitive Closure
4

Negative Cycles

After finishing, if any diagonal element dist[i][i] < 0, a negative cycle exists because the shortest path from a node to itself shouldn't be negative.
