Web APIs & Dependency Injection

Understand standard inversion-of-control life cycles to register services cleanly inside built-in dependency resolution trees:

  • Transient: Re-allocated freshly upon every request parameter call.
  • Scoped: Shared as a unique instance across an individual network context connection lifetime.
  • Singleton: Instantiated once globally at application bootstrap.
